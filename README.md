# RELOVE AI - Breakup Recovery Coach

A production-ready full-stack MVP that provides AI-powered breakup recovery coaching with rule-based responses and OpenAI integration.

## Features

- **Daily Action Selection**: Get personalized daily guidance based on your breakup recovery stage
- **SafeText Rewriter**: Transform risky messages into confident, attractive communication
- **Greenlight Check**: Determine if it's the right time to reach out to your ex
- **AI Agent Passthrough**: Direct access to OpenAI GPT-5 for complex situations

## Tech Stack

- **Backend**: Node.js + Express + TypeScript
- **Frontend**: React + Vite + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Testing**: Vitest
- **AI Integration**: OpenAI GPT-5 API
